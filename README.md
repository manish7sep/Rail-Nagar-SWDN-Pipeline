# Rail-Nagar-SWDN-Pipeline
Open-source Python pipeline for smart water distribution network analysis on Rail Nagar ward, Rajkot. Combines EPANET 2.2 with WNTR 1.4 for EPS, water-quality, leak-diagnostic, resilience, and sensor-placement analyses in 8 reproducible stages. MIT-licensed reference code for Bol et al. (2026), Journal of Hydroinformatics.
